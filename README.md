## 앱 개발 과정

충분히 할 수 있었고 너무나도 재미있는 과제였습니다. 사진 URL 기능과 사진 delete 기능만 좀 더 시간만 있었으면 구형했을 텐데라는 아쉬움이 있습니다.
계속 꾸준히 하니 앱이 작동하는, 보람찬 과정이었다고 생각합니다.
